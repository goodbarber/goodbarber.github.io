Unlock GoodBarber's Magic

Building GoodBarber would not have been possible without Open Source Software. We wanted to do our bit to give back to the community and here's some of that software. We hope you'll find it useful. For a full list, see our Github.
Want to make contributions? We'd love to see them! All we ask is that you adhere to our FOSS Community Code of Conduct to help us build a fair, inclusive and welcoming environment.

  • Custom Features
  • API integrations
  • Internal Libraries

  1. See a demo

    Links example

    This project shows you how you can make a link to an internal page of a GoodBarber Custom code section. It demonstrates also how to pass parameters to this internal page. To illustrate this, when the user type is name in the field, a new page is opened saying hello to that name.

  2. See a demo

    Requests example

    This project shows you how you can send a request to an external API using the GoodBarber App SDK. To illustrate this, when the user click on the button, a request is sent to the Cat Fact API.

  3. See a demo

    Storage example

    This project shows you how you can use storage with the GoodBarber App SDK. It's also an example of how you can implement a taking note feature in GoodBarber.

  4. See a demo

    User example

    Using the GoodBarber App SDK you can get informations about the current user logged into your application. This project shows you how to do it.

  1. See on Github

    plugin_demo

    This is a demo GoodBarber plugin that shows all the methods you can use to create your plugin for the GoodBarber engine.

  1. See on Github

    # GBRecyclerIndicator

    This library is used in the Android native engine of GoodBarber. It handles Android RecyclerViews items removing boilerplate code and automatically manage the different types of elements on the List.
